# Contributing to Mila

Thanks for your interest! Mila is a native macOS (Swift/SwiftUI) app built on
whisper.cpp, with optional on-device speaker diarization via pyannote.audio.

## Development setup

Requirements: macOS 14+, Xcode 15.3+, Homebrew.

```bash
make bootstrap   # installs xcodegen via brew if missing
make project     # generates Mila.xcodeproj from project.yml (the source of truth)
make build       # or `make run` to launch
make test        # unit tests
```

The Xcode project is generated by XcodeGen from `project.yml` — edit
`project.yml`, never the `.xcodeproj`. See [`CLAUDE.md`](./CLAUDE.md) for
architecture and conventions.

## Pull requests

- Branch from `main` and open a PR; all PRs require review before merging.
- Keep changes focused and explain what and why (the PR template prompts you).
- Run `make build` and `make test` locally first — CI must be green.
- **Never commit secrets, credentials, tokens, or internal infrastructure
  details.** Push protection is enabled and will reject obvious secrets.
- Note any user-facing change in the README changelog.

## Bugs and feature requests

Use the issue templates. For security issues, follow
[SECURITY.md](./SECURITY.md) — do **not** open a public issue.
